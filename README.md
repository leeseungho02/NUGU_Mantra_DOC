# What is Mantra

## 목적

스트레스로 인해 지치고 우울증에 시달리는 현대인들의 스트레스를 해소하고, 건강한 정신을 유지하도록 가이드

## 서비스 핵심 내역

* 명상 음악을 출력하여 명상을 지도
* 수면 유도 4-7-8 호흡법 지도
* 공부/업무에 필요한 집중력을 상향시켜주는 백색소음 출력
* 사용자의 결과적 통계를 기반으로 음악 추천
